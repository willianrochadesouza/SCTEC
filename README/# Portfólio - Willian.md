# Portfólio - Willian  
  
Este projeto foi desenvolvido como parte do programa SC Tec (Carreira Tec), com o objetivo de apresentar minha evolução e os conhecimentos adquiridos na área de desenvolvimento de software.  
  
## Sobre o projeto  
  
Este portfólio reúne conceitos iniciais de desenvolvimento web, utilizando HTML, CSS e noções básicas de JavaScript.    
A proposta foi aplicar, na prática, os conteúdos vistos durante o curso, organizando informações de forma clara, estruturada e com uma identidade visual própria.  
  
Além disso, o projeto também integra minha experiência prévia com design, contribuindo para a construção do layout e da experiência visual da página.  
  
## Tecnologias utilizadas  
  
- HTML (estruturação de conteúdo)  
- CSS (estilização e layout)  
- JavaScript (conceitos básicos de interação)  
- VS Code (ambiente de desenvolvimento)  
- GitHub (hospedagem do projeto)  
  
## Aprendizados aplicados  
  
Durante o desenvolvimento deste projeto, foram aplicados conceitos como:  
  
- Estruturação de páginas web  
- Organização de conteúdo e seções  
- Estilização com CSS (cores, espaçamento e layout)  
- Noções básicas de lógica de programação  
- Interações simples com JavaScript  
- Uso de ferramentas de desenvolvimento  
  
## Sobre o desenvolvimento  
  
Ao longo do curso, tive contato com a criação de elementos como formulários, interações básicas, lógica simples e componentes visuais.  
  
Este portfólio representa a aplicação prática desses conhecimentos, mesmo sendo um projeto inicial, e marca o início da minha jornada na área de desenvolvimento.  
  
## Objetivo  
  
Meu objetivo é evoluir continuamente na área de tecnologia, desenvolvendo soluções que unam lógica, design e aplicação prática no mercado.  

## Observação sobre o uso de IA

Durante o desenvolvimento deste projeto, utilizei ferramentas de apoio como o GitHub Copilot para auxiliar na construção de algumas partes em JavaScript.

O uso dessas ferramentas teve como objetivo otimizar o processo de desenvolvimento e apoiar o aprendizado, sempre buscando compreender e adaptar os códigos gerados conforme os conceitos estudados.
